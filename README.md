# Holiday Manager

## Overview
The Holiday Manager manages holidays for you and your team. It has two main features as follows:

1. Dashboard
2. Tracker

### Dashboard
This is where you can approve the holidays of those who report to you and see a summary of your own holidays.

### Tracker
This is where you can see the holidays of all your team members.
